Sublime Text 3 安装及Python环境搭建

发表于 2017-08-29 | 阅读次数:

Sublime Text 3 是一个轻量级的跨平台文字编辑器，尤以其轻快的速度，易用性和强大的社区支持而著称。它一经面世便被认为是一个杰出的编辑器，而它真正的强大之处在于你可以使用 Package Control 来增加它的功能以及可以进行各种自定义设置.

1.官网下载：http://www.sublimetext.com/3

2.破解

选项 help > license ，将以下所有文字粘贴到提示框内。

3.安装package control（https://packagecontrol.io/ ）

提醒：装完了所有插件都要重启软件才可以生效

1)通过快捷键 ctrl+` 或者 View > Show Console 菜单打开控制台。

2)粘贴如下代码（https://packagecontrol.io/installation）：

SUBLIME TEXT 2

SUBLIME TEXT 3

4.安装插件

1). 按下Ctrl+Shift+P调出命令面板。

2). 输入install 调出 Install Package 选项并回车。

3). 在列表中选中要安装的插件，或者输入插件名(比如要安装IMESupport插件，则输入IMESupport，它会实时过滤筛选结果)，根据命令面板中的过滤结果，选择要安装的插件。

4).常用插件
sublimecodeintel 代码自动补全(在某类型代码环境下，html,css,js等)
emmet 快速的html/css编写神器

5.Python运行环境（安装 SublimeREPL）

Ctrl+shift+p 键入 install packages

稍等片刻后 键入 SublimeREPL 安装即可
